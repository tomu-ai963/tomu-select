# tomu-select

tomu select — AIプロダクト販売 + 開発サービス依頼のB2Bハブ

白基調・ミニマル・ゴールドアクセントのフルページUI。依存なしの単一HTMLファイル(`index.html`)で構成されています。

## 表示方法

`index.html` をブラウザで開くだけで動作します(ビルド不要・外部依存なし)。

## 構成

- **ナビゲーション** — スティッキーヘッダー、スクロールスパイ(現在地ハイライト)、モバイル幅ではハンバーガーメニュー
- **Hero** — 見出し + CTA(Browse Products / 開発を依頼する)
- **Product Grid** — JSレンダリングの3カラムカード(モバイル1カラム)。カードクリックで詳細モーダルを表示
- **商品詳細モーダル** — フル説明・機能リスト・価格。「導入相談する」でServicesへスクロール + 該当カテゴリタグを自動選択
- **Services** — カテゴリタグ + 問い合わせフォーム(必須項目バリデーション、送信後は完了メッセージに遷移。実API送信は未実装)
- **フッター** — リンク群 + コピーライト

## 実装メモ

- 商品データは `index.html` 内の `PRODUCTS` 配列にハードコード。将来API取得に差し替える場合は、`renderSkeleton()`(スケルトンローディング表示)→ fetch → `renderProducts()` の順で呼び替えるだけで移行できる構造
- カラー・シャドウはCSSカスタムプロパティでトークン化。ライト/ダークテーマ対応(`prefers-color-scheme` + `data-theme` 属性の両対応)
- `prefers-reduced-motion` 対応(スムーススクロール・シマー・トランジション停止)
- フォーム送信はフロントのみの状態遷移。バックエンド接続は今後の課題
